# Proyecto EV3 

Este es la evaluación de aprendizaje número 3 del curso de Programación Orientada a Objetos (POO)

El proyecto consiste en el desarrollo de una aplicación que permita la gestión de usuarios mediante un sistema de registro e inicio de sesión, utilizando una base de datos MySQL para almacenar la información de los usuarios; las contraseñas están cifradas para garantizar la seguridad con la librería bcrypt.
